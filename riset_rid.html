Kuesioner System Usability Scale (SUS)

Responden (manajer) akan menilai 10 pernyataan berikut dengan skala 1 (Sangat Tidak Setuju) hingga 5 (Sangat Setuju).

  1. Saya berpikir akan sering menggunakan sistem ini.
  2. Saya merasa sistem ini terlalu rumit.
  3. Saya merasa sistem ini mudah untuk digunakan.
  4. Saya pikir saya membutuhkan bantuan teknis untuk dapat menggunakan sistem ini.
  5. Saya menemukan berbagai fungsi dalam sistem ini terintegrasi dengan baik.

(dan 5 pernyataan lainnya...)
